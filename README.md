## How to compile

### Option A — Overleaf (recommended, zero setup)
1. Go to [overleaf.com](https://overleaf.com) → **New Project → Upload Project**
2. Upload `resume.tex`
3. Set compiler to **pdfLaTeX** (default)
4. Click **Recompile** → download PDF

### Option B — Local (requires TeX Live / MiKTeX)
```bash
pdflatex resume.tex
```

## What to fill in before compiling

Search for `[` in `resume.tex` — every bracketed item is a placeholder:

| Placeholder | What to put |
|---|---|
| `[Start Year]` | Year you joined Ashley Furniture |
| `[Senior / Lead Title]` | Your actual previous job title |
| `[Company Name]` | Your previous employer |
| `[Year] -- [Year]` | Employment date ranges |
| `[N]` / `[X]%` | Real numbers from your work |
| `[BE / BTech, Field]` | Your undergraduate degree |
| `[Institution Name]` | Your undergraduate college |
| `[Year of Completion]` | MTech graduation year |
